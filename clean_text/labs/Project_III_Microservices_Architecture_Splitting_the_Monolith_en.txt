Software Construction Project Type: Group Work / Final Project Topic: Deconstructing the monolithicThesis Management System intoa Microservices Architecture.
Epic / Project Vision
The existing monolithic Thesis Management System is successful, but it does not scale well for the future. If the notification code fails, the entire system crashes. The Vision: The system will be decomposed into at least three independent microservices. Each service is responsible for its own domain, has its own database (Database-per-Service pattern), and can be deployed independently.
